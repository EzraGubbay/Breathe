Claude-Code-specific notes:

- The Xcode project is generated by **XcodeGen**. After creating, deleting, or
  renaming any source file you MUST run `xcodegen generate` before building.
  Never edit `Breathe.xcodeproj` directly (it's gitignored).
- Verify behavior with the XCUITest smoke tests, not by clicking the Simulator
  window — synthetic mouse clicks against the watch simulator are unreliable
  from this environment, and `simctl` cannot send taps.
- `xcrun simctl io "Breathe SE2" screenshot <path>` works fine for visual checks.
